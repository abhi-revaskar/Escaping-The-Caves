Group Name :- DECODERS
----------------------------------------------
Steps to run the Makefile:
Enter the command "make attack" in the terminal and press enter. This will run below 3 files.
1)"cipher_to_message.py" - this file converts the hexadecimal string to a message. This message is printed on the terminal.
2)"root_generation.sage" - this file generates the root which is required to find out the password to clear this level. The value of root is printed on the screen.
3)"root_to_password.py" - this file generates the password which is required to clear this level. The password is printed on the screen.
